## Назначение директорий

`cmd/metopt-lab1` — точка входа в приложение. Здесь находится main.go, который нужен для запуска программы и объединения всех частей проекта.

`internal/objective` — пакет для целевых функций оптимизации, включая функцию Розенброка.

## Реализация модели конструктивных действительных чисел

Конструктивное действительное число представляется не как просто сохранённое значение, а как объект, который умеет выдавать приближение с заданной точностью epsilon.

Это выражено через интерфейс:

```
type Real interface {
    Approx(eps float64) (float64, error)
}
```

Таким образом, любое число в данной модели должно поддерживать получение приближённого значения по запросу.

### Что реализовано

- интерфейс Real;
- создание константного конструктивного числа через NewConstant;
- создание конструктивного числа из функции приближения через NewFromApprox;
- арифметические операции:
  - Add;
  - Sub;
  - Mul;
- сравнение чисел через Compare;
- обработка некорректного значения epsilon;
- тесты для базовой функциональности, арифметических операций и сравнения.
